Evaluación Completa de Expresiones Matemáticas

1. Evaluación con variable z = 4.5

Variable definida: z = 4.5

(a) Expresión algebraica

0.424 + 3.122 - 162.32 - 80.7

(b) Expresión con raíz cúbica

(z3 - 23) / (∛(z2 + 17.5))

2. Evaluación con variable t = 3.2

Variable definida: t = 3.2

(a) Expresión exponencial

½ e2t - 3.8 t3

(b) Expresión racional

(6t2 + 6t - 2) / (t2 - 1)

3. Evaluación con variables x = 6.5, y = 3.8

Variables definidas: x = 6.5, y = 3.8

(a) Expresión combinada

(x² + y²)2/3 + (xy)/(y - x)

(b) Expresión con raíz cuadrada

√(x + y)/(x - y)² + 2x² - xy²

4. Evaluación con variables derivadas

Variables definidas:

c = 4.6, d = 1.7

a = c·d², b = (c + a)/(c - d)

(a) Expresión exponencial compleja

[e(d - b) + ³√(c + a) - (c·a)d] / b

(b) Expresión algebraica combinada

d/c + (a/b)² - ed - a/b

5. Verificación de Identidades Trigonométricas

Sustituyendo x = π/10 (18°):

(a) Identidad trigonométrica básica

cos²x - sin²x = 1 - 2sin²x

(b) Identidad con tangente

tanx/(sinx - 2tanx) = 1/(cosx - 2)
